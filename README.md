<h1 align="center">Welcome to closest-pair-doubling 👋</h1>

> The implementation of the Randomized O(n log n)-time Closest-Pair Algorithm

### [Homepage](https://github.com/ThangMinhCao/closestpairdoubling)

### [Demo](https://)

## Prerequisites

- npm >=5.5.0
- node >=9.3.0

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
